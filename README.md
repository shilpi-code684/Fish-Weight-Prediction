# Fish Weight Prediction Project


---

# 📌 Overview

This project is a Machine Learning based Fish Weight Prediction System developed using Python and Scikit-learn. The system predicts the **weight of fish** based on physical characteristics such as length, height, and width measurements.

The project demonstrates the complete Machine Learning workflow including:

- Data Collection
- Data Preprocessing
- Feature Selection
- Model Training using Linear Regression
- Prediction System
- Accuracy Evaluation

This project was created as part of academic coursework at **Bakhtiyarpur College of Engineering**.

# 📊 Dataset Description

The dataset contains various fish species with the following features:

| Feature | Description |
|---|---|
| **Species** | Species name of the fish |
| **Weight** | Weight of fish in grams (Target Variable) |
| **Length1** | Vertical length in cm |
| **Length2** | Diagonal length in cm |
| **Length3** | Cross length in cm |
| **Height** | Height in cm |
| **Width** | Diagonal width in cm |

# 📈 Model Performance

The trained Linear Regression model evaluates performance using:

- **Mean Absolute Error (MAE)**: Average prediction error in grams
- **R² Score**: Coefficient of determination (0-1 scale, higher is better)
